# 第五天，mongodb数据库

安装
sudo apt install -y mongodb-org
sudo apt install mongodb

mongo -version 查看版本

/etc/mogodb.conf 进入配置文件

sudo service mogodb start 启动mongodb

ps ajx lgrep mongodb 查看进程

sudo service mongodb stop 结束进程

db 查看当前数据库名称
db.stats() 查看当前数据库信息
exit 退出连接

mongo 启动客户端服务
